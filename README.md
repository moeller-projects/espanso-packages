# espanso-packages

Espanso packages for AI-agent workflows.

## Packages

- `agent-engines` — semicolon-prefixed triggers that route prompts to the right AI engine.
- `agent-prompts` — pipeline-driven prompts for implementing work items, reviewing PRs,
  debugging, refactoring, and authoring decision artifacts.

## Notes

- `agent-prompts` is the newer package. The legacy `agents` package will be removed in a
  future commit, so update any references to use the new package name.
